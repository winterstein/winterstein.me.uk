                                    : If the image has a stable url somewhere else then you can hot-link to it.

                                    : 
                                    : 1. Go to [assets](/assetManager)
                                    : 2. Upload the image you want to use. *PLEASE RESIZE IT FIRST!* 
                                    : You are allowed to upload up to 10mb, which would be a bad idea!
                                    : 3. Now go to the text editor for your blog post. You should see a thumbnail of your image at the bottom of the page.
                                    :  4. Drag and drop your image into the text editor. That will insert the source url, but not an image
                                    : tag. You need to convert the url into an `img` tag, adding width, height and other settings if you like.
                                    : It's good practice to add alt and title attributes (alt is for blind people and whilst the image loads, 
                                    : title is shown on mouseover - usually they should be the same). E.g. convert
                                    : 
                                    : `http://everyoneisplaying.com/static/uploads/1/0073555delmee__p8882863010604442581.jpg`
                                    : 
                                    : into
                                    : 
                                    : <code>
                                    : <img src="http://everyoneisplaying.com/static/uploads/1/0073555delmee__p8882863010604442581.jpg"
                                    : width="200" alt="Labyrinth by Paul Delmee" title="Labyrinth by Paul Delmee">
                                    : </code>

 How to make blog posts             | To post:
                                    : 
                                    :  - login (duh)
                                    :  - go to "write" in the nav bar
                                    :  - this should lead you to an editor which is hopefully clear enough to use (please feel free to suggest improvements - we are also putting in support for posting-by-email).
                                    :  - enter your post and hit publish
                                    : 
                                    : Great - the post now exists and can be viewed, but it isn't listed on the front page. To do that:
                                    : 
                                    :  - go to "settings" in the nav bar and set the editor controls to "some extra features"
                                    :  - now go to your post, which you can find via your profile, and at the bottom you should find an "edit" button.
                                    :  - This takes you back to the editor but with a few more controls
                                    :  - In the "tags" field, enter a comma-separated list of tags including "group:frontpage"
                                    :  - You can also set the description - which is the blurb that will be shown on the front page. It defaults to grabbing the 1st paragraph of the post.
                                    :  - now save, and the post should now be listed on the front page.
